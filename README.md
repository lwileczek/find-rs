# find-rs
Unix Find written in Rust

You're probably looking for [fd](https://github.com/sharkdp/fd)

## Features
 - Respects .gitignore files
 - Searches in parallel
 - Memory safe :)

## Status 
Alpha

## Getting Started
### Build  
`cargo build`  
### Run  
```bash
./find-rs <glob> <directory>
```
